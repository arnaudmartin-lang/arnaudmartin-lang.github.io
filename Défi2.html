Défi2:Code source

Pour trouver ce red flag, nous avons inspecté le code source en faisant un clic droit. Il y avait le mot de passe qui était écrit.
Red flag: HTML=langage-du-web!
Pour éviter que des personnes découvrent votre mot de passe il faut mettre en œuvre des techniques de sécurisation dès le départ car on peut trouver des informations sensibles et les failles web sont souvent dues aux erreurs de programmation, de configuration, ou d'oubli de sécurisation.
Il y a comme risque professionnelle:
Fuite de données sensibles (Data Leakage)
Le code source est interprété par le navigateur de l'utilisateur, ce qui signifie que tout ce qui s'y trouve est public. En milieu professionnel, cela peut révéler :
Des commentaires de développeurs contenant des notes sur la structure interne du réseau.
Des identifiants de test (logins/mots de passe) oubliés.
Des clés d'API ou des jetons de session.
