Défi 8 : Une partie de Tétris

Pour trouver le flag on est rentré dans le code source puis on a recherché le nombre (999999999). Après avoir trouvé, on copie le code qui nous amène sur une page avec le flag.

Flag : j4v4scr!pt=cot3_cli3nt

JavaScript est un langage de programmation utilisé pour les animations, jeux et 3D. Son exécution est locale, ce qui signifie que le code peut être modifié côté client.

Pour éviter les hacks, il faut protéger le code source afin que seul l’admin puisse modifier les valeurs.

Risques : - Accès non autorisé au code source - Vol de données sensibles - Vulnérabilités applicatives
