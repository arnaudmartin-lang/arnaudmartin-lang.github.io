Javascript est un langage de programmation très utilisé pour faire des animations sur les sites Web, des jeux et même de la 3D ! Son exécution est locale : cela veut dire que c'est ton ordinateur qui exécute le programme... Cela veut aussi dire que tu peux le modifier pour changer son comportement !
Pour ne pas se faire hacker il faudrait protéger ton code source pour que personne à part l’admin puisse changer les valeur dans le code
Il y a comme risque professionnelle:
Risques de sécurité informatique
Accès non autorisé au code source → peut permettre d’exploiter des failles ou de voler des données.
Exposition de données sensibles (codes, clés, informations internes).
Vulnérabilités applicatives si le code est modifié sans contrôle.
