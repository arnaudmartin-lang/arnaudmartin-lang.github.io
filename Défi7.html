Défi7 : Initiation à JavaScript

Pour trouver le flag, il faut aller dans le code source puis chercher une phrase en JavaScript.

Une fois trouvée, on la copie et on la valide dans Challenges Kids.

Flag : JS_C0d3_Sh4ll_b3_S3cured!

JavaScript est un langage qui permet de rendre les pages web interactives et dynamiques.

Quand une page HTML contient du JavaScript, le code est téléchargé et visible côté client.

Il est donc important de sécuriser son code pour éviter les abus ou le vol d’informations.

Risque professionnel : Faille XSS (Cross-Site Scripting)

Si une application accepte du JavaScript injecté par un utilisateur malveillant, celui-ci peut : voler les cookies, détourner la session ou rediriger vers des sites frauduleux.

Exemple de code dangereux : <script>alert(1);</script>
