Défi7 : Initiation à Javascript

Pour trouver le flag il faut aller dans le code source puis trouver une phrase en Javascript. Après avoir trouvé le code on le copie et on le rentre dans challenges kids.

flag : JS_C0d3_Sh4ll_b3_S3cured!

Javascript est un langage qui permet de rendre les pages web interactives et dynamiques. En téléchargeant une page HTML qui contient du javascript, on peut lire le code source et comprendre comment la page fonctionne.

Il est donc très important de protéger son code source pour éviter que les pirates le volent et détournent l'application à des fins malveillantes.

Pour éviter les vols de données, il faut éviter de mettre des mots de passe dans les phrases en Javascript car on peut facilement les retrouver.

Il y a comme risque professionnel :
Risque de piratage des utilisateurs (Faille XSS)

Si l'on se fie au bout de code <script>alert(1);</script> présent sur l'image, le risque associé (le XSS) est dévastateur. Si l'application permet à un utilisateur malveillant d'injecter du JavaScript, l'attaquant peut voler les cookies de session, rediriger vers des sites frauduleux, ou modifier l'apparence du site.
